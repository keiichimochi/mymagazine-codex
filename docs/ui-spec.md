# UIデザイン仕様

## デザイン方針

90年代ゲーム雑誌の情報密度、ハード戦争感、編集部らしさを持つWebツールにする。

## 画面構成

- 左側: 編集部ヒアリングフォーム
- 右側: 生成された世界線エンジン
- 下部: 16ページ構成、DBプレビュー、生成パイプライン

## トーン

- 技術オタ誌
- BEEP的
- ゲーメスト的な熱量
- ただし操作UIは現代的に読みやすくする

## カラー

- 背景: 紙面に近い淡いグレー
- メイン: 黒、赤、青、黄色の雑誌見出し色
- アクセント: 警告帯、速報帯、レビュー点数欄を想起する色

## コンポーネント

- テキスト入力: 複数行回答用
- ボタン: サンプル投入、生成、JSONコピー
- タブ: 概要、16ページ、DB、パイプライン
- カード: ライター、ハード、記事枠などの個別要素のみ

## レスポンシブ

- デスクトップでは2カラム。
- モバイルではフォーム、結果、ページ構成の順に縦積み。
- テキストがボタンやカードからはみ出さないように折り返す。
